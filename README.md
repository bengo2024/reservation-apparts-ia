# Réservation Appartements Meublés + IA Générative

Application web (React) + mobile (React Native) avec intelligence artificielle pour :
- Recommandations personnalisées  
 Chatbot conversationnel  
 Analyse automatique des photos  
 Prédiction dynamique des prix  
 Détection de fraudes

## Stack technique
- Backend : Node.js / NestJS ou Python FastAPI
- Frontend Web : React + TypeScript + Tailwind
- Mobile : React Native (iOS & Android)
- Base de données : PostgreSQL + Redis
- IA : Python + TensorFlow/PyTorch + modèle LLM (Llama 3 ou Mistral)
- Cloud : AWS ou GCP
- Paiement : Stripe
- Déploiement : Docker + GitHub Actions + Render/Vercel/Fly.io

## Fonctionnalités principales
- Recherche avancée avec filtres et carte
- Réservation instantanée + paiement sécurisé
- Recommandations IA personnalisées
- Chatbot intelligent 24/7
- Tableau de bord propriétaire et admin
